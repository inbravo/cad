What is Scribe?

Scribe is a stateless and standalone web based microservice for managing CRM objects using simple REST calls. Scribe hides the CRM specific complexities from your application. Another benefit of Scribe is for CRUD operations on CRM objects. You can create CRM object like Account/Case/Lead etc using REST calls; which in turn are created at respected CRM. Currently Scribe is developed with an intention of integration with Sales Force, NetSuite, Microsoft Dynamic XRM, Zoho and ZenDesk

Main feature

  • Talk to all CRMs in a generic CRM agnostic language
  • Use same communication mechanism for all CRM types
  • Avoid internal CRM complexity by simply using XML/JSON
  • Decouple your CRM integration layer
  • Add new CRM integrations without affecting your native systems
  • Supports following CRM
    • SalesForce
    • NetSuite
    • MS Dynamics XRM
    • Zoho
    • ZenDesk ticketing system
