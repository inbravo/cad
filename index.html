What is CAD?

CAD is a stateless and standalone web based microservice for managing CRM objects using simple REST calls. CAD hides the CRM specific complexities from your application. Another benefit of CAD is for CRUD operations on CRM objects. You can create CRM object like Account/Case/Lead etc using REST calls; which in turn are created at respected CRM. Currently CAD is developed with an intention of integration with Sales Force, NetSuite, Microsoft Dynamic XRM, Zoho and ZenDesk
