What is Scribe?

Scribe is a web based stateless microservice for managing CRM objects using simple REST calls. Scribe hides the CRM specific complexities from your application. You can perform all crud (create/read/update/delete) operations on respective CRM objects like Account/Contact/Case/Lead/User/Ticket etc. Every Scribe action results in objects created/read/updated/deleted at respective CRM

Main feature

  • Talk to all CRMs in a generic CRM agnostic language
  • Use same communication mechanism for all CRM types
  • Avoid internal CRM complexity by simply using XML/JSON
  • Decouple your CRM integration layer
  • Add new CRM integrations without affecting your native systems
  • Supports following CRM
    • SalesForce
    • NetSuite
    • MS Dynamics XRM
    • Zoho
    • Zendesk
